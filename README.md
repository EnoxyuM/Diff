##  How to Use

#### 1. To Apply a Patch (`Paste Diff`)
1. Paste text into the **left window**.
2. Copy content of `diff` file (including headers like `--- a/file.txt`) to your clipboard.
3. Click **`Paste Diff`** button. The patched result will be in the right window.

#### 2. To Create a Patch (`Copy L2R Diff`)
1. Fill left window with Original text and right window with Changed text.
2. Click  **`Copy L2R Diff`** button. Diff file to change Left text(Original) to Right(Changed) will be copied to your clipboard.

#### 3. To Find difference (`Find Diff`)
1. Just click da button...

#### 4. Delete comments (`DelCom`)
This button copy left to right window and deletes comments in left window ignoring inside text("// /**/") and in regex.
It also replaces strings that contains only spacebars with empty strings.
